# Install-PsBuildTask

## SYNOPSIS
Install PsBuildTasks task file.

## SYNTAX

```
Install-PsBuildTask [-Path] <DirectoryInfo> [-Task] <String> [<CommonParameters>]
```

## DESCRIPTION
Downloads a task file and saves it to the tasks directory.
